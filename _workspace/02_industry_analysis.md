**본 분석가의 사이클 위치 재판단 (fundamental 데이터 반영)**:
메모리 매출 QoQ +101.6% / YoY +292%는 **사이클 상승 중기에서 후기로 가속**되는 강도. 2Q 비트그로스 가이던스가 mid-single이라는 점은 (a) 추가 capa 가용성 제약, (b) 의도적 공급 통제 가능성 — 어느 쪽이든 **ASP 채널이 비트그로스를 압도하는 구도** 확인. HBM4E 2Q 첫 샘플은 SK하이닉스 추격 시그널이며, "2027 주문 캐파 초과" 코멘트는 본 보고서 Bull 시나리오 트리거 일부 실현으로 해석.

**ASP·마진 가정 시나리오 (fundamental 권고 반영 — 2026-05-12 2차 상향)**:

fundamental의 검증 의견 ("Base 35%는 슈퍼사이클 국면을 과소반영, 상향 권고")을 수용하여 **메모리 OP 마진 가정을 2차 상향**:

| 시나리오 | 1차 가정 | **2차 (최종, fundamental 권고)** | 사이클 위치 |
|---------|----------|-------------------------------|------------|
| Bull (Peak 26~27) | 50%+ | **55~60%** | 1Q26 실측 65%+ 보수, 27년 캐파 부족 지속 |
| Base (Mild Peak→평준화) | 42% | **40~45%** | 26 평균: 1H26 60%+, 2H26 50%대 점진 둔화 |
| Bear (정점 조기 종료) | 30% | **25~30%** | 정점 1Q26→2H26 빠른 둔화, 평준화·다운 초기 |

**비교 대조 (현 시점 메모리 OP 마진 = 역사적 정점급)**:
- SK하이닉스 1Q26 OP 마진 **72%** (Digitimes 2026-04-23 / SK하이닉스 IR)
- 마이크론 1Q26 (CQ1) OP 마진 **67.6%** (ZDNet, tradingkey 인용)
- 삼성 DS 1Q26 통합 OP 마진 = 53.7 / 81.7 ≈ **65.7%**, 메모리 단독은 **70%대 추정** (산식 비공개)
- 2018 슈퍼사이클 정점 메모리 OP 마진 50%대 (참고치) → **현 사이클은 2018 대비 마진 강도 +15~20%p 상회**

**컨센서스와의 갭**:
- FnGuide FY26 OP 컨센 **120~170조 원** (증권사 격차 큼)
- 1Q26 실측 OP 57.2조 × 4 단순 = **228.8조** → 컨센 평균(145조)을 56% 상회
- 키움 2Q26 100조 전망 적용 시 연간 200조+ 시나리오 가능
- → **본 보고서 Base 시나리오(메모리 마진 40~45%)는 컨센 평균과 정합**, Bull(55~60%)은 키움·SK증권(₩500k 목표가) 시나리오와 정합

**fundamental SOTP 적용 노트** (회신 자료 인용):
- fundamental팀이 보고서 멀티플 시나리오에 적용한 가정: **2026E 연간 평균 마진 Bull 50% / Base 40% / Bear 28% + 메모리 매출 280~320조**
- 본 보고서 산업 점수표는 "연간 평균 마진" 기준으로 운용 — fundamental 가정과 정합. **분기 정점 마진(60%대)을 산업 평가에 그대로 적용하지 않음**.

### 1.3. 사이클 리스크

- **CXMT(중국)의 DDR5 진입**: 2026년 본격 공급 가능성이 가격 상단의 잠재 압력.
- **HBM3E → HBM4 전환기 가격 디버전스**: HBM3E는 2H26 YoY 하락 압력, HBM4는 상승 (TrendForce, 2025-12-18).
- **빅테크 FCF 압박 → CapEx 집행률 저하 리스크** (macro팀 인풋 2026-05-12): 2026E 합산 CapEx $705~725B의 자유현금흐름 압박이 누적되어 집행률 80% 시나리오가 현실화될 경우, 메모리 발주가 2H26부터 둔화될 수 있음. 본 분석가 견해로는 Base 시나리오의 'ASP 횡보'가 이 채널을 통해 발생할 가능성이 가장 높다.

## 8. 지정학 / 공급망 리스크

### 8.1. 주요 사안

| 사안 | 발생 가능성 | 영향도 | 영향 채널 | 영향 시점 |
|------|------------|-------|---------|----------|

| CHIPS Act 후속 / Taylor TX 팹 보조금 (-26% 감액) | **확정·진행 중** | 中 | 파운드리 캡티브 + Tesla AI6 | 2026~2027 |
| 대만 해협 긴장 (TSMC 의존도) | 낮음~중 | 高 | 全세계 IT 공급망 | 비정형 |
| 일본 EUV 포토레지스트 수출 정책 | 낮음 | 中 | 첨단노드 수율 | 잠재 |
| CXMT(중국) DDR5 진입 | 중 | 中 | DRAM ASP 상단 | 2026 2H~ |

### 8.2. 시안 NAND — 단기 안정, 중기 불확실 (macro팀 2026-05-12 확인)

- **2025-12-30**: 미국 상무부가 삼성·SK하이닉스·TSMC 중국 fab에 **2026년 연간 장비 수출 라이선스 승인** (출처: CNBC/Reuters/Tom's Hardware)
- **2025-12-31**: VEU(Validated End-User) 면제 시스템 만료 → **건별 면제에서 연간 라이선스 체계로 전환**
- 삼성은 시안에서 **전세계 NAND의 ~40%** 생산 (사실)
- 운영·유지보수 지속 가능, 단 첨단 설비 확장은 제한
- **EUV는 여전히 금지 품목** (NAND는 EUV 불필요, 단 향후 멀티패터닝 한계 시 영향 가능)
- 2026 1~5월 사이 추가 통제·완화는 큰 변동 없음

**시나리오 반영**:
- "운영 지속" → Base 시나리오 (단기 OP 충격 회피)
- "연간 재승인 리스크" → Bear 시나리오 (NAND 공급 unhedge 변수)

### 8.3. CHIPS Act / Taylor TX Fab (macro팀 2026-05-12 확인)

| 항목 | 내용 |
|------|------|
| 확정 보조금 | 기존 $6.4B → **$4.745B로 26% 감액** (2025년 말 조정) |
| 삼성 전체 미국 투자 규모 | $44B → **$37B 축소** |
| 텍사스 주정부 인센티브 | +$250M (2025-09 발표) |
| 연방+주정부 합계 | 약 **$5B** |
| 건설 진척률 | 90%+ (2025년 말 기준) |
| 장비 반입 (Phase 1) | **2026년 초 시작** |

| 두 번째 fab | 2027년 |
| 기술 노드 | 당초 4nm → **2nm 업그레이드** |
| 핵심 수주 | **Tesla AI6 $16.5B FSD 공급** — "no customers" 우려 해소 |

### 8.4. 평가

지정학은 **단기 호재(라이선스+CHIPS 집행)·중기 불확실성(연간 갱신·보조금 축소)·장기 기회(Taylor 가동) 혼재**. 본 분석가 견해로는 NAND 시안 라인은 향후 12개월 매출 가시성에 영향이 없으나, 24~36개월 CapEx 배분 의사결정에 큰 영향 — 보고서의 장기 시나리오에 반영 필요.

### 정량 근거 핵심

**(1) 메모리 슈퍼사이클**:
- 1Q26 실현 OP **₩57.2조** vs 키움 추정 ₩31조 vs 컨센 ₩27.6조 — **컨센이 실제를 큰 폭 언더슈트**
- 모바일 DRAM/NAND **실제 +90% QoQ** (Counterpoint) vs 증권사 추정 +45% — TrendForce 가이드(+55~60%)보다도 실제 더 강함
- FY26 OP 컨센 분포: FnGuide ₩140조 ~ 씽크풀 ₩170조 ~ **KB ₩360조** — 1Q×4 단순 ₩228조에 평균이 못 미침

**(2) HBM4 NVIDIA 30%+**:
- 4/1~5/6 주가 +59.09%, SK하이닉스(+104.96%) 갭 좁히기 = HBM 디스카운트 빠른 축소
- KB ₩360,000, CLSA/미래에셋 ₩400,000은 HBM 35%+ 시나리오 반영
- "Strong Buy" 컨센(Buy 36/37) = HBM 듀얼 벤더 내러티브 전환 완료

**(3) Galaxy S26 / MX 마진**:
- 1Q OP ₩57.2조 = 메모리 단독으로 MX 부담 완전 흡수 → 시장이 MX 리스크를 "메모리 호황의 그림자"로 인식
- 사전예약 +10%는 표면적 호재로 인식, BOM 상승 충격 가려짐

**(4) 2nm 파운드리**:
- 가동률 80%+, 4Q26 흑전 가시화로 파운드리가 "악재→옵션"으로 전환
- 키움 "2Q OP 100조" 상향 논리에 파운드리·S.LSI 흑전 명시 = 부분 반영 시작
- 퀄컴 70% 수율 조건 미충족 — 적극 흑전 시나리오는 미반영

### 산업 점수표 적합성 평가

본 보고서의 점수 가중치는 sentiment 검증과 **모순 없음**:
- **호재 (+) 내러티브 ((1)·(2))**: 시장 반영도 50~85%로 일부 가격 발견 진행. 1Q26 실적이 컨센을 능가 → **상향 사이클 진행 중**. 본 보고서가 호재(强)/호재(中)으로 평가한 가중치는 정합.
- **악재 (−) 내러티브 ((3)·(4))**: 시장 반영도 10~55%로 미반영 영역. 본 보고서가 MX 악재(中)·파운드리 중립으로 평가한 가중치는 **현 시점 시장 컨센서스에 노출되지 않은 리스크** → 통합 시 Bear 시나리오 트리거로 활용 권장.
