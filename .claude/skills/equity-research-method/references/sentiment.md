# Sentiment — 시장 심리·수급·컨센서스 분석 가이드

심리 애널리스트가 시장 참여자 포지셔닝과 컨센서스를 평가할 때 사용하는 표준.

## 1. 증권사 컨센서스

### 수집 항목
- **목표주가 분포**: 평균, 중앙값, 최대, 최소
- **투자의견 분포**: 매수(Strong Buy/Buy) / 유지(Hold) / 매도(Sell/Underweight)의 비율
- **EPS 컨센서스**: 현재 회계연도, 다음 회계연도
- **매출 컨센서스**
- **OP 컨센서스**

표준 형식:

| 항목 | 평균 | 중앙값 | 최대 | 최소 | 표준편차 | 표본 수 |
|------|------|-------|------|------|---------|---------|
| 12M 목표주가 | ₩X | ₩Y | ₩Max | ₩Min | σ | N개 기관 |
| FY+1 EPS | | | | | | |
| FY+1 매출 | | | | | | |

### 컨센서스 변화 추이
- **1개월/3개월 전 대비** 상향 비율 vs 하향 비율
- **어닝 모멘텀 지수**: 상향-하향 차이를 표본수로 나눈 값

출처: 에프앤가이드, 와이즈에프엔, 블룸버그, S&P Global Market Intelligence

## 2. 수급 분석

### 모니터링 지표
- **외국인 누적 순매수** (5일/20일/60일/120일)
- **외국인 지분율** (현재 vs 1년 전)
- **기관 누적 순매수** (연기금/투신/금융투자/보험 등 세부 주체별 가능)
- **개인 누적 순매수**

### 표준 표 형식
| 주체 | 5일 | 20일 | 60일 | 120일 |
|------|-----|------|------|-------|
| 외국인 | | | | |
| 기관 | | | | |
| 개인 | | | | |

(단위: 천억 원 또는 만 주, 명시)

출처: KRX KIND, 금융감독원 전자공시

## 3. 파생 / 공매도 / ETF

### 공매도
- **공매도 잔고 수량 / 비율**
- **공매도 일별 추이**
- 출처: KRX 공매도 종합 포털

### 대차 잔고
- 대차 잔고는 공매도 압력의 선행 지표
- 출처: 금융투자협회

### ETF 편입
- **KODEX 200, TIGER 200 등 주요 ETF의 삼성전자 비중**
- **MSCI 한국 ETF의 비중**
- **글로벌 반도체 ETF (SMH, SOXX 등)의 편입 여부 및 비중**

### 옵션 / 선물 (한국 시장은 KOSPI200 옵션이 주류)
- 개별주 옵션은 한국에서 활성화되지 않음. 코스피200 옵션의 풋콜 비율, VKOSPI(코스피200 변동성지수) 등을 시장 전반의 심리 지표로 활용

## 4. 뉴스 / 이벤트

### 분류 카테고리
| 카테고리 | 예시 |
|---------|------|
| 실적 | 잠정실적, 가이던스, 어닝 서프라이즈/쇼크 |
| 사업 | 신제품 출시, 대형 계약, M&A, CapEx 발표 |
| 거버넌스 | 자사주 매입, 배당, 분할/합병, 지배구조 |
| 인사 | CEO/CFO 교체, 사내이사 변경 |
| 사고/소송 | 화재, 데이터 유출, 특허 분쟁 |
| 정책/지정학 | 수출 통제, 정부 보조금, 무역 분쟁 |
| 외부 평가 | 신용등급 변경, ESG 평가 |

### 표 형식
| 날짜 | 카테고리 | 요약 | 주가 반응 (당일/주간) | 출처 |
|------|---------|------|---------------------|------|
| YYYY-MM-DD | | | | |

최근 3개월의 주요 이벤트 5~15건 수록 (중요도순).

### 주가 반응 측정
- **당일 반응**: 발표 당일 종가 변동률
- **5거래일 누적 변동률**
- **비교군**: KOSPI 또는 SOX 대비 초과수익률 (특수 이벤트 영향을 분리)

## 5. 거버넌스 / 정책

- **자사주 매입 / 소각 발표**
- **배당 정책 변경**
- **지배구조 이슈**: 이재용 회장 관련 소송, 지주사 전환 논의 등
- **노조 / 임단협**: 파업 가능성, 임금 인상률 합의

각 이벤트의 발표 시점 + 시장 반응을 함께 기록.

## 6. 정성 시그널 (커뮤니티 / SNS)

### 사용 방침
- 정량 결론 도출에 사용하지 않는다
- 출처를 명시할 수 있는 경우만 인용
- 익명 토론방·확인되지 않은 트윗은 보고서에 포함하지 않는다

### 활용 예시
- "X 플랫폼에서 HBM3E 수율 관련 우려가 반복적으로 제기되었으나, 회사 공식 자료에서는 확인되지 않음 → 검증 필요 사안으로만 기록"

## 7. 종합 심리 진단

다음 3단계 평가:
- **과열 (Overheated)**: RSI 70+ + 외인 4주 연속 순매수 + 목표주가 상향 우세 + 거래량 급증
- **중립 (Neutral)**: 명확한 신호 부재
- **과매도 (Oversold)**: RSI 30 이하 + 외인 매도 정점 + 컨센서스 하향 우세

진단 결과에는 **판단 근거 3개 이상**을 명시한다 (단일 지표 진단 금지).
