# Citations & Disclaimers — 인용·면책 표준

모든 애널리스트와 통합 에이전트가 따르는 출처 표기·면책 규정.

## 1. 인용 우선순위

### 1차 자료 (최우선)
- **DART 전자공시시스템** (대한민국 금융감독원) — 공시 원문
- **회사 IR 자료** — 분기/연간 실적 발표 자료, 컨퍼런스 콜 트랜스크립트
- **거래소** — KRX, NASDAQ, TWSE
- **중앙은행 / 정부 통계** — BOK ECOS, FED FRED, KOSIS, 산업통상자원부
- **글로벌 시장조사기관 공식 발표** — TrendForce, Gartner, IDC, Counterpoint, Omdia (보고서 또는 보도자료)

### 2차 자료 (1차 자료로 역추적 후 인용)
- 증권사 리서치 보고서 — 가능하면 핵심 수치의 1차 자료를 확인 후 인용
- 주요 언론 — 로이터, 블룸버그, 닛케이, 한경/매경, 조선/중앙 등
- WSTS / SIA 통계

### 3차 자료 (인용 자제)
- 익명/검증 불가 SNS 글
- 비공식 블로그
- 위키피디아 (보조 참고만)
- 루머 / 비공식 채널 정보 → 보고서 본문에 포함 금지

## 2. 인용 표기 형식

### 정량 데이터
```
[수치] (출처: [발표기관], [발표일 또는 데이터 시점])
```
예: "분기 매출 76.8조 원 (출처: 삼성전자 2024 Q4 잠정실적공시, 2025-01-31)"

### 정성 정보 / 인용문
```
[내용] (출처: [매체명], [기사 제목 또는 보고서명], [발표일])
```
예: "사장이 HBM3E 양산 일정 단축을 언급함 (출처: 회사 2024 Q4 컨퍼런스 콜, 2025-01-31)"

### 표 인용
표 하단에 통합 출처를 표기:
> Source: KRX (수급), 에프앤가이드 (컨센서스), 회사 IR (실적). 데이터 시점: YYYY-MM-DD.

## 3. 단위·통화 표기

- 한국 회계 단위: **조 / 억 / 백만 원** — 보고서 내 일관 사용 (혼용 금지)
- 환율 환산이 필요한 경우 환산일 환율 명시 — "USD 환산 시 1 USD = ₩1,395 (2025-MM-DD 매매기준율)"
- 비율: 소수점 첫째 자리 또는 둘째 자리까지 일관 사용

## 4. 데이터 시점

- 모든 시점은 **YYYY-MM-DD** 형식
- 분기는 **FY2025 Q1** 또는 **2025 1Q** 표기 (한 보고서 내 통일)
- 시점 미확정 정보(예: 향후 발표 예정)는 "발표 예정"으로 명시, 추정 수치 금지

## 5. 면책 조항 (필수 포함)

모든 최종 보고서 말미에 다음 면책 조항을 포함한다:

```markdown
## 면책 조항 (Disclaimer)

본 보고서는 공개된 정보를 바탕으로 작성된 **정보 제공 목적의 분석 자료**이며,
특정 주식의 매수·매도·보유를 권유하는 투자권유 자료가 아닙니다.

- 본 보고서에 포함된 수치, 전망, 시나리오는 작성 시점의 공개 정보 및 분석 가정에 기반하며,
  미래의 실제 결과와 다를 수 있습니다.
- 투자 결정은 본인의 판단과 책임 하에 이루어져야 하며, 본 보고서의 작성자/배포자는
  본 정보를 활용한 투자 결과에 대해 책임을 지지 않습니다.
- 본 보고서에는 컨센서스(증권사 평균 전망) 및 분석 가정에 기반한 추정치가 포함되며,
  실제 데이터와 차이가 있을 수 있습니다. 핵심 의사결정 시 1차 자료(공시·재무제표)와
  최신 시장 데이터를 별도 확인할 것을 권장합니다.
- 본 보고서의 작성자는 본 종목과 관련하여 이해관계가 없음을 명시합니다.
  (해당이 다른 경우 명시 필요)
```

## 6. 데이터 한계 명시

보고서 부록에 다음 형식으로 한계를 명시한다:

```markdown
## 부록 B. 데이터 시점 및 한계

- 본 보고서의 데이터 수집 시점: YYYY-MM-DD
- 실시간 시세 데이터: [확보 가능 / 일부 제한 / 불가] 명시
- 컨센서스 데이터 출처: [명시]
- 비공개 사업부문 영업이익 등 추정 사용 영역: [명시]
- 시점 불일치가 남아있는 데이터: [있다면 명시]
```

## 7. 금지 사항

- ❌ 임의 추정값을 사실로 가장
- ❌ 출처 없이 수치 인용
- ❌ 단정적 가격 예측 ("X원이 될 것이다")
- ❌ 단정적 추천 ("반드시 매수하라")
- ❌ 검증 불가능한 SNS / 루머 인용
- ❌ 단위 / 통화 / 시점의 혼용
- ❌ 작성 시점에 이미 공개된 사실을 누락한 채 시나리오 구성
