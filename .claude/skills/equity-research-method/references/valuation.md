# Valuation — 펀더멘털·밸류에이션 방법론

펀더멘털 애널리스트가 적정주가를 산출할 때 참조하는 표준 절차.

## 1. 멀티플 기반 밸류에이션

### 사용 멀티플
- **PER (Price / EPS)** — 가장 직관적. 경기민감주는 사이클 정점에서 낮고, 바닥에서 높게 나오는 함정에 유의.
- **PBR (Price / BPS)** — 자산 헤비 기업에 적합. 삼성전자 분석 시 ROE와 함께 본다 (ROE 높으면 PBR 프리미엄 정당화).
- **EV/EBITDA** — 자본 구조에 중립적. 글로벌 동종업계 비교 시 표준.
- **PSR (Price / Sales)** — 적자 또는 마진 변동성이 큰 시기에 보조 지표로.

### 산식
```
적정가 (멀티플) = 타깃 멀티플 × 12개월 Forward EPS(또는 BPS, EBITDA, 매출)
```

타깃 멀티플은 다음 중 하나로 결정한다:
- 과거 5~10년 평균 또는 중앙값
- 동종 글로벌 경쟁사 평균
- 사이클 정점/바닥 멀티플 (특정 시나리오)

### 한국 회계기준 주의
- 지배주주순이익 vs 연결순이익: 보고서 일관성을 위해 **지배주주순이익** 기준 EPS 사용
- 일회성 손익 정상화: 충당금/일회성 이익 차감 후 정상 EPS 산출 (정상화 사유와 금액을 노트에 명시)

## 2. SOTP (Sum-of-the-Parts)

다각화된 기업에 적합. 삼성전자는 다음 부문으로 분해 가능:

| 부문 | 대표 비교 기업 | 적용 멀티플 | 비고 |
|------|--------------|------------|------|
| DS (메모리) | SK하이닉스, 마이크론 | EV/EBITDA, PER | 사이클 위치 반영 |
| DS (시스템LSI/파운드리) | TSMC | EV/EBITDA, PER | 첨단 노드 점유율 가정 |
| SDC (디스플레이) | LGD, BOE | EV/EBITDA | 분기별 부침 큼 |
| MX (모바일) | 애플(참고) | PER, EV/Sales | 마진 안정성 평가 |
| VD/CE | LG전자 가전사업 | PER, EV/EBITDA | |
| 하만 | Visteon 등 | EV/EBITDA | |
| 기타: 현금성 자산, 자회사 지분 | - | 장부가 또는 시가 | 디스카운트 5~30% 적용 |

각 부문의 적정 EV 합 → 순부채 차감 + 비영업자산 가산 → 적정 시가총액 → 발행주식수로 나눠 적정주가.

**홀딩 디스카운트** 또는 **컨글로머릿 디스카운트** 10~25%를 적용하는 것이 일반적이다 (적용 시 사유 명시).

## 3. 간이 DCF (참고용)

자세한 DCF는 가정값 수가 많아 본 하네스의 범위에서는 **간이 버전**만 사용한다:

1. 향후 5년 영업현금흐름 추정 (전년 대비 성장률 가정 명시)
2. 5년 후 영구성장률 가정 (한국 명목 GDP 성장률 근처, 1.5~2.5%)
3. WACC 산출:
   - 무위험수익률 = 한국 10년 국채 금리
   - 베타 = 1.0~1.3 (대형주 추정 범위, 출처 명시)
   - 시장 위험 프리미엄 = 5.0~7.0% (한국 ERP 추정치, 출처 명시)
4. NPV = ∑(FCF_t / (1+WACC)^t) + Terminal Value / (1+WACC)^N
5. 순부채 차감, 발행주식수로 나눔

DCF는 가정 민감도가 크므로 결과를 **단일 값이 아닌 ±20% 범위**로 제시한다.

## 4. 자본 효율성 지표

| 지표 | 산식 | 평가 기준 |
|------|------|---------|
| ROE | 지배주주순이익 / 평균 자본총계 | 대형 IT는 10~20% 수준이 평균 |
| ROIC | NOPAT / 투하자본 | WACC 대비 스프레드가 핵심 |
| 순부채/EBITDA | (총차입금 - 현금성자산) / EBITDA | 음수면 순현금 보유 |
| 배당수익률 | DPS / 주가 | 한국 대형주 2~3% 일반적 |
| 배당성향 | DPS / EPS | 25~30% 수준 |

## 6. 산출 결과 통합

펀더멘털 적정가는 3개 방법(멀티플·SOTP·간이 DCF)을 산출하고, **각각의 가정과 함께 표로 제시**한다. 단일 값으로 합치지 말고 범위로 보여준다:

| 방법 | 가정 (요약) | 적정주가 |
|------|-----------|---------|
| 멀티플 (Forward PER 12배) | EPS 컨센서스 X원 | ₩A |
| SOTP | 부문별 멀티플 + 25% 디스카운트 | ₩B |
| 간이 DCF (WACC 8.5%, g 2%) | 5년 OP CAGR 5% | ₩C~D |

리더가 이 표를 기술적 시나리오·컨센서스 목표주가와 함께 통합한다.
