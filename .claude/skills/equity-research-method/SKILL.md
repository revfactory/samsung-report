# Equity Research Method — 주식 가치 분석 방법론

한국 대형주(삼성전자 005930 등) 분석에 필요한 공통 방법론. 각 애널리스트 에이전트는 자신의 영역에 해당하는 references/만 선택적으로 로드한다.

## 언제 어떤 reference를 로드하나

| 작업 | 로드할 reference |
|------|----------------|
| 펀더멘털·밸류에이션 | `references/valuation.md` |
| 기술적 분석 | `references/technical.md` |
| 반도체 산업·경쟁 분석 | `references/semiconductor-industry.md` |
| 거시·외부 환경 분석 | `references/macro.md` |
| 컨센서스·수급·심리 분석 | `references/sentiment.md` |
| 출처/단위/면책 규정 | `references/citations-and-disclaimers.md` |

분석 영역 외 reference는 로드하지 않는다 — 컨텍스트를 절약한다.

## 모든 영역에 공통으로 적용되는 원칙

### 1. 사실 / 추정 / 의견 3단 구분
모든 문장은 다음 중 하나로 분류 가능해야 한다:
- **사실(Fact)**: 1차 자료(DART 공시, IR 자료, 거래소 데이터, 정부 통계)에서 출처와 함께 인용 가능한 정보
- **추정(Estimate)**: 컨센서스, 가이던스, 회사/3자 전망. 반드시 "컨센서스(에프앤가이드)" 등 출처 표기
- **의견(Opinion)**: 분석가의 판단. "본 분석가 견해로는", "본 보고서의 시나리오에서는" 등 명시

라벨이 모호한 문장은 보고서에 포함하지 않는다.

### 2. 단위와 시점 명시
모든 수치는 단위와 시점을 동반한다:
- 매출/이익: "47.2조 원 (FY2024 Q4, 잠정실적공시)" — 한국 회계기준은 "조/억 원" 단위 권장
- 가격: "₩72,500 (2025-MM-DD 종가, KRX)"
- 환율: "₩1,395/$ (2025-MM-DD, 한국은행)"
- 비율: "외인 지분율 53.x% (2025-MM-DD, KRX)"

분석 기준일이 다르면 가장 최근 일자로 통일하거나, 비교표에 각 시점을 명시한다.

### 3. 출처 표기
1차 자료 우선, 2차 자료(증권사 리포트, 언론)는 가능하면 1차 자료로 역추적 후 인용.
검증 불가능한 출처(익명 트윗, 미확인 루머)는 보고서에 포함하지 않는다.

상세 규칙: `references/citations-and-disclaimers.md`

### 4. 데이터 미확보 시 처리
실시간 데이터 접근이 불가능하거나 자료를 확인할 수 없는 경우, **임의 수치 생성은 절대 금지**한다. 대신:
- "마지막 확인 가능 시점은 [공식 공시·정기보고서 등]" 명시
- "최신 데이터 확인 필요" 라벨 부착
- 정성 평가로 보완 (가능한 범위 내)

이 원칙은 보고서의 신뢰도를 결정한다. 추정으로 사실을 가장하는 것이 가장 큰 품질 훼손이다.

### 5. 가격 단정 회피
"X원이 될 것이다" 형식의 단정 표현을 피한다. 대신:
- "현재 PER 12배 가정 시 적정가 X~Y원 범위"
- "{조건} 충족 시 Bullish 시나리오의 1차 저항은 X원"
- "12개월 컨센서스 목표주가 평균 X원 (범위 Y~Z원, N개 기관)"

### 6. 시나리오 작성
시나리오는 항상 **트리거 조건 + 가정값 + 결과 범위** 3요소를 갖는다.
- 트리거: "DRAM 현물가 4분기 연속 상승"
- 가정값: "2026년 메모리 OP 마진 35%, PER 14배"
- 결과: "적정가 X~Y원 범위"

가정이 누락된 시나리오는 검증 불가능하므로 작성하지 않는다.

### 7. 면책 / 비투자권유 명시
모든 최종 보고서에는 면책 조항을 포함한다. 템플릿은 `references/citations-and-disclaimers.md`.

## 통합 시 데이터 일관성 규칙

여러 에이전트가 같은 지표를 다르게 인용한 경우의 처리:

1. **시점 통일** — 가장 최근 공통 일자로 맞춤. 불가하면 표 형식으로 시점별 병기
2. **출처 추적** — 두 값의 출처가 다를 경우 1차 자료(공식 통계·공시) 우선
3. **단위 통일** — 조/억/백만 원의 혼용 금지, 보고서 전체에서 단위 일관
4. **삭제 금지** — 상충하는 두 값을 그냥 하나로 결정하지 않고 출처와 함께 병기 후 통합자가 노트 부착

## 보고서 작성 톤

- 한국어 보고서로 작성. 기업명·지표명·종목코드는 영문/숫자 병기 (예: 삼성전자(005930), TSMC(2330.TW))
- "~합니다"체보다 "~하다"체(논문/리포트체) 사용
- 마크다운 표 적극 활용 — 가독성과 통합 용이성
- 결론 → 근거 순서로 배치 (탑다운)

## 데이터 수집 전략 (필요 시)

이 스킬은 데이터를 가져오는 도구를 제공하지 않는다. 각 에이전트가 다음 방식 중 적절한 것을 선택한다:

1. **공식 공개 자료 검색** — DART, IR 페이지, 한국은행 ECOS, KRX, 정부 통계 등 인용 가능한 1차 자료
2. **시장조사기관 데이터** — TrendForce, Gartner, IDC, Counterpoint 등 — 출처와 발표일 명시
3. **언론/증권사 리포트** — 1차 자료로 역추적 후 인용

웹 접근이 불가능한 환경에서는 보고서 본문에 "현재 환경에서 실시간 데이터 접근 불가 — 마지막 확인 가능 시점: X" 명시 후 그 시점 기준으로 분석 진행.
